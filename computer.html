Computer Science

Programming, Hardware, and Networking

Chapter 1: Computer Fundamentals

Basic components and operation of computers.

1.1 Hardware Components

  • CPU: Central Processing Unit - the brain
  • RAM: Random Access Memory - temporary storage
  • Storage: HDD/SSD - permanent storage
  • Input/Output devices: Keyboard, mouse, monitor

1.2 Software Types

  • System software: Operating systems
  • Application software: Programs for specific tasks
  • Programming software: Tools for coding

Chapter 2: Programming Basics

Introduction to programming concepts.

2.1 Data Types

  • Integers: Whole numbers
  • Floats: Decimal numbers
  • Strings: Text data
  • Boolean: True/False values

2.2 Control Structures

if (condition) { execute code }
for (initialization; condition; increment) { loop }

Chapter 3: Networking

Connecting computers and sharing resources.

3.1 Network Types

  • LAN: Local Area Network (home/office)
  • WAN: Wide Area Network (city/country)
  • Internet: Global network of networks

3.2 Network Protocols

TCP/IP, HTTP, FTP - rules for data communication.
